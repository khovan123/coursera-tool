Coursera Tool

Browser extension chạy cục bộ để tự động phát video và có thể mở quiz cho người dùng làm thủ công.

Dự án không có thanh toán, subscription hoặc pay-as-you-go.

Tải bản mới nhất Xem mã nguồn

Đang kiểm tra release mới nhất...

Cài từ file release

  • Bấm Tải bản mới nhất để tải file zip từ GitHub Release.
  • Giải nén file zip vừa tải.
  • Mở chrome://extensions và bật Developer mode.
  • Chọn Load unpacked và trỏ tới thư mục đã giải nén.

Build thủ công

  • Clone repository và chạy npm run build tại thư mục gốc.
  • Load unpacked thư mục apps/extension/dist.

Kết nối mạng

  • Coursera: content script chỉ thao tác trên trang đang mở.
  • Không gọi dịch vụ AI để tạo câu trả lời.
  • Không có payment API, quảng cáo hoặc analytics.
